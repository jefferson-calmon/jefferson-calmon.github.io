Proffy


A Proffy é uma plataforma que conta com site na web e aplicativo mobile para android e ios onde professores podem dar aulas e os alunos podem estudar, tudo online e muito pratico. Projeto desenvolvido durante a Next Level Week da Rocketseat

Principais tecnologias usadas

Algumas das tecnologias utilizadas:



  • NodeJS
  • ReactJS
  • React Native
  • Typescript
