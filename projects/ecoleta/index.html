Ecoleta

Ecoleta é um projeto criado na semana Next Level Week. Este é um evento criado pela Rocketseat e tem o objetivo de levar os desenvolvedores para o proximo nível

Sobre o projeto

Ecoleta é um marketplace de coleta de resídous. Ele conecta pessoas que querem descartar o lixo da maneira certa com conpanhias de coletas de resídous e reciclagem.
O projeto foi todo desenvolvido em JavaScript/TypeScript

Tecnologias

Para desenvolvermos este projeto foi necessário as seguintes tecnologias:

  • Node.js
  • ReactJS
  • React Native
  • Expo
